## 编写须知

- 图片请一定使用相对路径，所有图片放置于 `./pics` 下。

+ 部分常用函数，如三角函数、对数函数等的 Roman 体符号已经内置于 LaTeX 中。如余弦函数应当使用 `$\cos$` 表示，渲染为 $\cos$。而不应当使用 `$cos$` 表示，渲染为 $cos$。
